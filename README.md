- qTarget (albo jakiś inny target)
- ESX.ShowNotification
- PolyZone



## 🪛 Komendy 

Dostępne komendy:

```
    /tetrisGame
```

## ❤️ Export też dostępny! 

Przykładowe użycie exporta

Tetris:
```
    tetris/client.lua: 
        exports('tetrisGame', tetrisGame)
```
Twój kod:
```
    Twój/client.lua:
        exports['lrk_tetris']:tetrisGame()
```
